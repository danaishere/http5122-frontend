#Front-end Class Fall 2025 
This course covers "front-end" web technologies (including HTML, CSS, and JavaScript) to create interactive web pages. The main focus will be on programming concepts and structures in JavaScript.
## Tech Requirement(s)
The main software used during the course is [Visual Studio Code](https://code.visualstudio.com/)
